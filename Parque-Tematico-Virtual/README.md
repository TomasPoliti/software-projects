# Virtual Theme Park Management System

This project was developed as part of Software Design coursework. It consists of a management system for a virtual theme park, built while applying software architecture principles and development best practices.

## Overview

The project focused on designing and implementing a structured system with both backend and frontend components, emphasizing maintainability, clear architecture, and good software engineering practices.

## Technologies

- C#
- .NET 8
- Angular
- Entity Framework Core
- SQL Server

## Main concepts applied

- Layered architecture
- SOLID principles
- GRASP principles
- Domain modeling
- Clean Code
- Test Driven Development

## What I learned

Through this project, I gained experience with:
- backend development using C# and .NET
- frontend development with Angular
- persistence with Entity Framework Core
- relational databases with SQL Server
- separation of responsibilities across layers
- applying software design principles in a larger academic project

## Notes

The original repository for this project is private because it belongs to a university organization. For that reason, the source code is not included in this public repository.
