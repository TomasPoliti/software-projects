# TaskTrackPro

TaskTrackPro is an academic project developed for Software Design coursework. The goal of the project was to build a project and task management application while applying modern software engineering practices.

## Overview

The system was designed to support project organization, task tracking, and structured software development processes. The project emphasized both technical implementation and good engineering practices.

## Technologies

- C#
- .NET 8
- Blazor Server
- Entity Framework Core
- SQL Server

## Main concepts applied

- Layered architecture
- Object-oriented programming
- SOLID principles
- Design patterns
- Clean Code
- Test Driven Development

## What I learned

Through this project, I strengthened my understanding of:
- Backend development with C# and .NET
- Full application structure using layered architecture
- Persistence with Entity Framework Core
- Relational database integration with SQL Server
- Writing cleaner and more maintainable code
- Applying TDD in an academic project

## Notes

The original repository for this project is private because it belongs to a university organization. For that reason, the source code is not included in this public repository.
